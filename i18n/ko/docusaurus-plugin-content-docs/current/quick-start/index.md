## ⚠️ 중요사항

:::warning 영어 사용
**더 나은 결과를 위해 프롬프트는 영어로 작성해야 합니다.**

❌ 잘못된 예: `파란 다이아몬드 검`  
✅ 올바른 예: `blue diamond sword`
:::

:::tip 간단하게 시작
복잡한 설명을 시도하기 전에 "diamond sword" 같은 간단한 프롬프트로 시작하세요.
:::
